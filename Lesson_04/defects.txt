ID: DEF-001
Название: Некорректная обработка пустого символа в методе contains
Описание: Метод StringUtils.contains возвращает True, если передан пустой символ (""), 
что противоречит ожидаемому поведению теста. Тест ожидает False для пустого символа.
Шаги для воспроизведения:
  1. Создать объект StringUtils: utils = StringUtils()
  2. Вызвать utils.contains("SkyPro", "")
Ожидаемый результат: False
Фактический результат: True
Приоритет: Средний
Рекомендация: Добавить проверку на пустой символ в начале метода:
    if not string or not symbol:
        return False


ID: DEF-002
Название: Метод capitalize не обрабатывает None
Описание: Если в метод capitalize передан None, происходит ошибка TypeError, так как None не имеет метода capitalize().
Шаги для воспроизведения:
  1. Создать объект StringUtils: utils = StringUtils()
  2. Вызвать utils.capitalize(None)
Ожидаемый результат: ""
Фактический результат: TypeError
Приоритет: Средний
Рекомендация: Добавить проверку:
    if not string:
        return ""

Ожидаемый результат: ""
Фактический результат: AttributeError
Приоритет: Средний
Рекомендация: Добавить проверку:
    if not string:
        return ""
